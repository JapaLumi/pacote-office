Excel

O Excel é um software de planilhas eletrônicas. Ou seja, é um programa que utiliza tabelas para realizar cálculos ou apresentar dados. Com ele, é possível realizar cálculos matemáticos de forma automatizada, gerenciar a contabilidade de uma empresa, fazer relatórios, organizar dados em tabelas e até mesmo gerar recibos e notas fiscais.

Atalhos Função
Ctrl+A Abrir nova planilha.
Ctrl+B Salvar uma planilha.
Alt+C Vai para a página inicial.
Ctrl+W Fechar o documento.
Ctrl+C Copia um conteúdo selecionado.
Ctrl+V Copia o conteúdo que está na Área de transferência.
Ctrl+X Recortar algo da planilha.
Alt+S Ir para a guia de dados.
Alt+P Ir para a guia de layout.
Alt+K Ir para a guia exibir.
